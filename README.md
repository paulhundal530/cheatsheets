Personal, browser-viewable reference cheat sheets, served via GitHub Pages.

## System Design Patterns — Interview Cheat Sheet

Eight recurring patterns that cover almost every system design interview, plus the
fundamentals (data flow, scalability, fault tolerance, CAP), a failure-mode playbook,
security layers, and a closing scale checklist.

**View it live:** https://paulhundal530.github.io/cheatsheets/

Source: [`index.html`](index.html). Pattern material condensed from
[Hello Interview — System Design Patterns](https://www.hellointerview.com/learn/system-design/in-a-hurry/patterns);
the fundamentals / failure / security / checklist sections are standard-practice additions.
